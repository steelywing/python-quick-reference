## Basic
``` python
a, b, c = 1, 2, 3
# a = 1, b = 2, c = 3

1 < 2 < 3
# => True
```

### Bitwise Operations
``` python
~x # inverted
x | y # or
x & y # and
x ^ y # exclusive
x << n # shifted left by n bits
x >> n # shifted right by n bits
```
